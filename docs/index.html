Your best 5K effort this month was 24:18 — a 32-second PR. You're trending in the right direction. I'd suggest adding one more easy run per week to build your aerobic base further.

Quick Start
Running in 3 minutes
Install, configure your Strava credentials, and start talking to your data.

1. Install from PyPI

One command. No extra dependencies to manage.

2. Get Strava API credentials

Create an app at strava.com/settings/api and run python get_token.py

3. Add to Claude Desktop

Drop the config into your Claude Desktop settings and restart.

claude_desktop_config.json
{
  "mcpServers": {
    "strava": {
      "command": "strava-mcp",
      "env": {
        "STRAVA_CLIENT_ID": "your_id",
        "STRAVA_CLIENT_SECRET": "your_secret",
        "STRAVA_REFRESH_TOKEN": "your_token"
      }
    }
  }
}
